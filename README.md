# Shanghai Heritage Path Planning System

A Windows Forms application built with **ArcGIS Engine** that enables interactive path planning for Shanghai’s historic heritage points.

## Features
- **Map display** with TOC, toolbar, and status bar.
- **Zoom tools** – centered zoom‑in/out (new custom implementation).
- **Travel mode** – choose **车行** (car) or **人行** (pedestrian). The routing algorithm uses different network weight attributes (`DriveTime` / `WalkTime`).
- **Path planning** – solves a shortest‑cost path using ArcGIS `TraceFlowSolver`.
- **Result overlay** – highlights the solved path on the map.
- **Floating info window** – shows total distance **and** estimated travel time.

## Quick Start
1. Install the required ArcGIS Engine runtime (version 10.x).
2. Open the solution `Route_of_Shanghai.sln` in Visual Studio 2022.
3. Build and run – the main form appears with the map and controls.
4. Use the **地图工具** menu to select zoom, pan, or travel mode, then click **开始规划** to compute a route.
